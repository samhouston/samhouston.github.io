The browser requests the script file. Meanwhile, the parser blocks and stops parsing the other HTML on your page. After some time the script is downloaded and subsequently executed. The parser continues parsing the rest of the HTML document. --> <!doctype html> <html> <head> <title>My Page</title> <!-- old practice --> <!--- If you make a reference to any kind of markup or DOM element in this script, you will get an error.--> <script type="text/javascript" src="least_helpful.js"></script> </head> <body> <div id="user-greeting">Welcome, user.</div> <!-- universal support and best practice --> <script type="text/javascript" src="old_standard.js"></script> <!-- not universal support --> <!--async loads scripts asychronously, whenever the browser has freed resources, guaranteeing unblocked execution --> <script src="nonstandard_aysnc.js" charset="utf-8" async></script> <!--async guarantees execution, but not order of execution.
